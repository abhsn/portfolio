Hired Dream Job

My latest work so far. My first team project. A platform for managing career. A single place where job candidates and employers can find jobs and employees respectively. I learned a lot from this project. Both soft skills and technical skills. By doing this project I learned what a leader is and how to become a leader. For technical skills I learned many new technologies like, Socket.io, MongoDB aggregation, Git in deepth, Redux just a bit.

Sources

Live Website Client Server

Features

  • Candidates and employers can search for jobs, employers and candidates respectively. Employers can post and update jobs. Candidates can apply to those jobs and employers will get notification in real time.
  • Both types of user can update their profile and candidates can post blogs to showcase their skills.
  • Candidates can apply to job. Employers will get notified and can view all applied candidates. They can also check thier resume. Then they can have a meeting with candidates.

Technologies
