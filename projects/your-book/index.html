Your Book

Used book reselling e-commerce application where buyer can book, buy item, seller can sell,advertise item and admin can delete, verify a buyer.

Sources

Features

  • Email and Google authenticaion using firebase. Users can also choose their account while creating account. By default buyer type will be selected.
  • Different dashboards for admin, user and seller.
  • Buyers can pay for their booked product using stripe.

Technologies

  • React
  • Firebase
  • daisyUI
  • Node.js
  • ExpressJS
  • MongoDB
